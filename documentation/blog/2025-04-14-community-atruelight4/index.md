## Windows Support

Since the launch of codename mts, the open source community has been wanting and requesting the same automation by mts on Windows.

Currently, mts's CLI version works on Windows with limited built-in extensions via WSL, or Windows Subsystem for Linux. While there is a limited internal beta in the works, there are several bugs and improvements that are being worked on to ensure codename mts has the same seamless experience as it does on Mac.

## ATrueLight4's MTS4Win

Our featured contributor, [ATrueLight4](https://github.com/ATrueLight4), shipped **MTS4Win**, a fix that improves how MTS handles Windows-specific file paths.
